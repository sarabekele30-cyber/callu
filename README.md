# Callu 📞

Callu is a modern, open-source alternative to Discord built for seamless communication. It allows users to create text channels, build real-time voice/video chat rooms, and manage custom dashboards.

## 🚀 Features

* **Real-time Voice & Video:** Powered by WebRTC for low-latency communication.
* **Modern Dashboard:** Intuitive settings panel to configure your user profile and chat rooms.
* **Cross-Platform Potential:** Includes structure templates for desktop applications.
* **Developer Friendly:** Fully written in TypeScript using Next.js and TailwindCSS.

## 🛠️ Tech Stack

* **Frontend/Backend:** Next.js (App Router), React, TypeScript
* **Styling:** TailwindCSS
* **Database/ORM:** Prisma / PostgreSQL
* **Real-Time Data:** WebRTC & WebSockets
